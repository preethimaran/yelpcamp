# Yelp Camp 
Yelp Camp is a definitive Fullstack Web Application built using Node.js, Express, MongoDB and Bootstrap to mimic the functionality of the Review PLatform Yelp. Users can register to create campgrounds and users who are logged in can leave reviews for the created campgrounds. This project was built as part of [@Colt](https://github.com/Colt)
